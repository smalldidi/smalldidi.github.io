命令：
clean 删除生成的文件和缓存
config 获取或设置配置
deploy 部署您的网站
generate 生成静态文件
help 获取命令帮助
init 创建一个新的Hexo文件夹
list 列出网站的信
migrate 将您的网站从其他系统迁移到Hexo
new 创建新帖子
publish 将草稿从_drafts移动到_posts文件夹
render 使用渲染器插件渲染文件
server 启动服务器
version 显示版本信息
全局选项：
–config 指定配置文件，而不是使用_configure.yml
–cwd 指定cwd
–debug 在终端中显示所有详细消息
–draft 显示草稿帖子
–safe 禁用所有插件和脚本
–silent 隐藏控制台上的输出
